# Roll Feathers

Companion app for [Pixel Dice](https://github.com/GameWithPixels).

## Features
- Track multiple dice
- Track roll history
- Roll with Advantage & Disadvantage
- Blink Dice on roll
- Home Assistant Light integration
- API to get rolls

- Support
  - Pixel Dice
  - GoDice
- cross-platform
  - Android
  - iOS
  - MacOs
  - Windows
  - Linux
  - Web (Chromium)
